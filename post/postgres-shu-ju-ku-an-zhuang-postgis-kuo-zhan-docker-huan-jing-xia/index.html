Postgres数据库安装PostGIS扩展（Docker环境下）

2024-12-25

  1. 进入容器：

    docker exec -it postgres bash

  2. 更新包列表并安装 PostGIS:

    apt-get update
apt-get install -y postgresql-12-postgis-3 postgresql-12-postgis-3-scripts

  3. 连接数据库并启用扩展:

    psql -U postgres

    \c postgres
CREATE EXTENSION postgis;
CREATE EXTENSION postgis_topology;
